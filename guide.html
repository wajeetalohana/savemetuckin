Overview

This page is a user-facing guideline describing the main app screens and flows.

Home Tab — What it contains

The Home tab is the main discovery screen. Typical elements include:

  • Welcome / Quick actions — greeting.
  • Listings feed — recent items posted by users within a radius (2 km)
  • Filters / Search bar — let users filter by keywords.
Typical user journey on Home:
  1. Open Home → see nearby or recent posts.
  2. Tap a post to view details (photos, description, price, expiry).
  3. From post details: message seller

Sending a message to any post

Messaging is private and tied to the post. Basic flow:

From Home or Post details:
  1. Open the post you are interested in.
  2. Tap Send Message.
  3. Type your message — introduce yourself, reference the post title, propose a time or ask a question.
  4. Send. The other user receives a notification .
Messaging constraints & good practices
  • Be polite and include the post title in your first message (helps seller manage replies).
  • If the app limits messages per day (e.g., 25/day) or auto-deletes messages after a time, you'll see a warning before sending.
  • Messages are stored under a private chat linked to the item.
From My Post (as poster):
  1. Open your inbox → view incoming messages from interested users.
Tip: As a user, you have the option to block anyone you find misbehaving. Once blocked, you will no longer see that person’s posts. You may also unblock them at any time.

My Post Tab — How to create an item

This tab lists items you personally posted and includes a button (+) to create a new item. Steps to create an item:

1) Locating & starting
  1. Open My Post tab.
  2. Tap the + Register Item button (or Create).
2) Fill in item details
  • Title: short, descriptive (e.g., "Baby stroller — good condition").
  • Description: more details, condition, how to collect.
  • Price: numeric value (or mark as free).
  • Photos: add image which shows item clearly.
  • Location: auto-detected or manual address (used for showing nearby results).
  • Expiry end date .
3) Register
  1. Tap Register to submit it for review.
  2. App shows confirmation; item appears under My Post and in Home feed of other user accounts after approved by admin.
Tip: Double-check photos and description before publishing.

Validator Tab — What it contains

The Validator When some post a listing then it went through validation process. First step is the validator. Any person can be validator and validate the post and decide to approve or reject. Admin can approve or reject validator decision.

Tip: Final decision will be made by Admin if post should be approved or not.

User Profile

Profile contains your personal information and steps to upgrading account to premium:

  • Display name — visible to other users in chats.
  • Email — used for login and notifications (verify email for trust).
  • Upgrade to premium — a button to upgrade account to use premium feature.
  • Delete account — a button to delete account.
How to upgrade
  1. Go to Profile → Upgrade.
  2. review price and terms.
  3. Complete payment through the platform's payment flow (App Store. Keep receipts for billing).
  4. After successful payment, account gets premium privileges immediately (or after a short verification process).

WG Bot — what this might mean & how it could work

"WG Bot" is a simple chat bot which can tell you joke or quote of the day. You can also ask simple things like post, crate item, block user...

Quick example text templates

Use these message templates when contacting a seller:

  • First contact: "Hi — I’m interested in your [item title] . Is it still available? I can pick up on [date/time]."
  • Follow-up: "Thanks — where should we meet? My phone: [phone number]"
  • Seller reply sample: "Yes, it’s available. I can hold it until [date]. Let me know when you’ll collect."
